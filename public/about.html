General Entrees recipe app delivers, short, simple, and delicious recipes with accompanying pictures and step-by-step cooking instructions. A search tool lets you discover recipes just right for a certain ingredient, cuisine, or occasion from our library of recipes! Don't worry, you can add you own favorite recipes as well. Recommendations, based on the current time of day, are displayed on the front page.

Story Board

• Should provide a place to store ingredient / portions, preparation instructions, # of servings, and a picture • Add/Update/Delete personal recipes in the app • Categorize recipes (ex dinner, dessert, appetizer, etc • Search by functionality (by recipe name, or category, or ingredient • Rate recipes or some type of social sharing or commenting • Ability to log into app as different users • Create a shopping list • Add recipes from the web (any url, or more likely a stand formatted API)

Technology

General Entree is a web application using a Node and Express Web Server, backed by a MySQL Database. Sequelize is being used as the interface between the two system (ORM). The application is deployed on a public cloud platforn, Heroku, for the Production envionment. To meet new technology requirements, Owl Carousel 2, a Touch enabled jQuery plugin, is being used to create the beautiful responsive carousel slider.
